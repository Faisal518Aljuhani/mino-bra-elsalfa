# مين برا السالفة 🕵️

لعبة استنتاج اجتماعي (شبيهة بلعبة Spyfall) مع نظام حسابات آمن بالبريد الإلكتروني، تشتغل على جهازك محلياً.

## المميزات
- تسجيل حساب بالبريد الإلكتروني + رابط تفعيل حقيقي يوصلك على إيميلك
- كلمات مرور مشفّرة (bcrypt، لا تُحفظ أبداً كنص عادي)
- حماية من هجمات تخمين كلمة المرور (قفل الحساب بعد 5 محاولات فاشلة + تحديد معدل الطلبات)
- جلسات دخول موقّعة بـ JWT
- استعادة كلمة مرور منسية عبر البريد
- غرف لعب حية بأصدقائك عبر كود غرفة (Socket.io)
- 20 فئة، كل فئة فيها 10 عناصر (سيارات، أكلات، أفلام، أندية رياضية...)
- لعبة "حرف، اسم، حيوان، نبات، جماد، بلاد" — حرف عشوائي ومؤقت دقيقة ونص، بوضعين: أونلاين (غرف، يحتاج حساب) وجهاز واحد (تمرير الجوال، بدون حساب). النقاط: ١٠ للإجابة الفريدة، ٥ للإجابة المكررة بين لاعبين، صفر للخطأ أو الفاضية.
- لعبة المافيا ولعبة العامل المشترك (أوضاع إضافية بدون حساب)

## المتطلبات
- تثبيت [Node.js](https://nodejs.org) (نسخة 18 أو أحدث) على جهازك

## خطوات التشغيل

### 1) ثبّت الحزم
افتح Terminal / cmd داخل مجلد `backend` ونفّذ:
```bash
cd backend
npm install
```

### 2) جهّز إعدادات البريد الإلكتروني
انسخ ملف `.env.example` باسم `.env` داخل مجلد `backend`، وعبّي القيم:

```bash
cp .env.example .env
```

أهم شيء تظبطه:
- `JWT_SECRET`: أي نص عشوائي طويل وسري (لا تشاركه مع أحد)
- `SMTP_USER` / `SMTP_PASS`: بيانات إيميل لإرسال رسائل التفعيل

**لو تستخدم Gmail:**
1. فعّل "التحقق بخطوتين" في حسابك
2. روح لـ https://myaccount.google.com/apppasswords وأنشئ "App Password"
3. حط الإيميل في `SMTP_USER` وكلمة المرور المؤقتة (16 خانة) في `SMTP_PASS`

### 3) شغّل السيرفر
```bash
npm start
```

بيطلع لك:
```
✅ السيرفر يشتغل على http://localhost:3000
```

### 4) افتح اللعبة
افتح المتصفح على: **http://localhost:3000**

سجل حساب جديد → تحقق من بريدك واضغط رابط التفعيل → سجل دخولك → أنشئ غرفة أو انضم بكود.

## ملاحظات أمنية
- قاعدة البيانات (`backend/db/app.db`) تُنشأ تلقائياً أول تشغيل — لا تشاركها مع أحد لأنها تحتوي بيانات المستخدمين.
- ملف `.env` يحتوي أسرار حساسة — لا ترفعه أبداً على GitHub أو أي مكان عام.
- الإعداد الحالي مخصص للتشغيل المحلي (على جهازك فقط). لو حبيت تنشره لاحقاً على الإنترنت عشان أصدقاء يلعبون من بعيد، تحتاج تستضيفه على سيرفر حقيقي (مثل Render أو Railway) وتفعّل HTTPS.
